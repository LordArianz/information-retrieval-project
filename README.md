# information-retrieval-project(GooGooli) 
Project of information retrieval course.

[front-end project](https://github.com/AOTOA/search-engine-ui)
## Project Structure:
### Offline Phase:
1. Parse documents
2. Clean documents
   1. Tokenize
   2. Normalize
   3. Lemmatize
   4. Remove stop words
3. Index documents
4. Build Language models
   1. unigram
   2. bigram
### Online Phase:
1. Query Check
   1. Spell Check
   2. Use language models
2. Rank documents
   1. tf-idf
   2. bm25
